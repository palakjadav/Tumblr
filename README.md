# Project 1 - Tumblr App

Tumblr is a an app used table views to display photo feed adds a new screen where the user can tap a Tumblr post to see details for that post. 
The purpose of this exercise is to practice passing data between view controllers.

Time spent: 5 hours spent in total

## User Stories

The following **required** functionality is complete:

- [x] Build the Main Photo Feed
- [x] User can pull to refres the main feed
- [x] Infinite scrolling to the main photo feed.
